# lando-hero-replica — fluid-reveal hero study

复刻 [landonorris.com](https://landonorris.com/) 首屏「流体头盔」效果的独立实现。
**零依赖、单文件**（`index.html`），无 CDN、无图片、无字体请求，所有视觉全部程序化生成，
双击文件或起任意静态服务器都能跑。

## 效果机制（逆向自官网，代码全部原创）

官网 hero 的四层管线，这里逐层对应实现：

1. **流体求解器（Stable Fluids / Jos Stam 1999）**
   在约 10% 屏幕分辨率的网格上跑：BFECC 半拉格朗日平流 → 鼠标外力注入（加性 splat）
   → 散度 → 4 次 Jacobi 泊松迭代 → 压力投影。关键参数对齐官网手感：
   `dissipation 0.972 · poisson×4 · dt 0.014 · 10% 分辨率`。
   泊松/散度/投影全部用 **2 格步长采样**——同位网格下抑制棋盘格失稳（官网同款做法）。
2. **背景噪声层**：双层 warp 的 fbm 值噪声渲到 1/4 分辨率 R16F RT，
   合成阶段在全分辨率做阈值化 + 按导数缩放的等高线描边（恒定 ~2px 细线）。
3. **头盔**：官网用 GLB 模型，这里用 **SDF raymarching 程序化建模**
   （椭球壳 + 平滑布尔挖出护目镜腔 + 下巴护板 + 尾翼 + 转轴盘 + 条纹涂装分区），
   包围球先做解析相交测试，没碰到的光线直接跳过，核显友好。
4. **合成**：速度场模长阈值化成"液体蒙版"——蒙版内调色板整体切换（米白纸面 → 灰绿）、
   头盔只在蒙版内显形；流体耗散后头盔重新"沉回纸下"。
   开场 1.15s 竖向卷帘 reveal，2.5s 无操作后假光标自动巡航（官网同款行为）。

## 操作

| 按键/行为 | 作用 |
|---|---|
| 移动鼠标 / 触摸拖动 | 画出液体轨迹，显形头盔 |
| 停 2.5 秒 | 自动巡航接管 |
| `C` | 切换配色（柠檬绿 lando ↔ 齐马蓝 zima） |
| `H` | 头盔常驻显示开关 |

`prefers-reduced-motion` 时关闭流体与巡航，头盔直接常驻。

## 调参

所有手感参数在 `index.html` 顶部 `CONF` 对象里；运行时可通过控制台 `HERO.conf.xxx` 热改，
`HERO.step(n)` 可在隐藏标签页里手动推帧（调试/截图用）。
